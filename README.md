# Boompy

## How to use it:
```
import boompy
boompy.set_auth({{account_id}}, {{account_username}}, {{account_password}})
acct = boompy.Account(name="test")
acct.save()
```

## Supported Entities
- Account
- Account Group
- Account Group Account
- Account User Role
- Environment
- Integration Pack
- Integration Pack Environment Attachment
- Role

## Supported API Actions
- getAssignableRoles
